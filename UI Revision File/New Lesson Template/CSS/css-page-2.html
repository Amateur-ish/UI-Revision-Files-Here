    CSS Selectors: How to manipulate HTML Element Styles

    CSS Selectors in CSS are the ones that select which elements affected by the CSS rules

    This is the basic syntax of CSS

    Selector - These are the HTML elements or elements that you want to style.

    Property -This is the attribute that you want to change (ex: color, font-size, background).

    Value - The value you want to apply to the property (ex: red, 16px).

    Types of CSS Selectors

    Element

    Selects all of the elements from the HTML.
    example:
    h1 {
    font-size:10px
    }

    Class Selector

    Selects all of the class selected from the HTML. Ends with a period
    example:
    .class {
    font-size:10px
    }

    ID Selector

    Selects the ID of an element from the HTML. Ends with a #
    example:
    #id {
    font-size:10px
    }
