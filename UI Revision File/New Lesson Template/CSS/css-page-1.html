    Designing the Web!

    CSS is about enhancing the looks and feel or visual presentation of a webpage by adding various design techniques. You can see CSS as the design for the HTML as it is used on all popular websites.

    What is CSS?

    CSS stands for Cascading style sheets. It means a stylesheet language that describes the presentation in terms of HTML or XML of a web page. CSS defines how elements should appear in terms of layout, colors, fonts, spacing, and other visual aspects.

    There are three types of implementing CSS:

    Internal

    Internal CSS are CSS inside the website. It is coded and worked in the HTML website itself.

    External

    External CSS are CSS that are imported outside of the HTML Website.

    Embedded

    Embedded CSS are CSS that are put directly from the element itself.
