    If-Else Statement and Logical Operators

    If Statements are Statements that only run when the conditions are met.

    Syntax:

    if(condition) {

    //Run Code Here

    }

    An Else Statements are Statements that only run when the condition is false

    Syntax:

    if(condition) {
    //Run Code Here
    }
    else {
    //Run Code here
